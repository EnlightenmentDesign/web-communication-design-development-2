Heading styled paragraphs

Paragraph styled as headings are used for stylistic purposes only.

If importance is meant to be shown, please use an actual heading tag as search engines will be able to translate correctly to the user.

Markup Example
<p class='h1'></p>

The little red fox jumped over the lazy dog
<p class='h2'></p>

The little red fox jumped over the lazy dog
<p class='h3'></p>

The little red fox jumped over the lazy dog

The little red fox jumped over the lazy dog

Markup Example

This line is meant to be treated as bolded text but without any emphasis.
